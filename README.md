# Store Sales Forecasting

## Project Overview

This project analyzes historical sales data from Corporación Favorita (Ecuador) and develops forecasting models to predict future sales.

## Business Questions

- Which product categories drive revenue?
- What is the impact of promotions?
- Do holidays increase sales?
- Which stores perform best?
- What factors influence future sales?

## Dataset

Store Sales - Time Series Forecasting (Kaggle)

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- XGBoost
- Power BI

## Project Structure

01_EDA.ipynb
02_FEATURE_ENGINEERING.ipynb
03_MODELING.ipynb
04_EVALUATION.ipynb

## Key Findings

- Grocery I contributes approximately 32% of total sales.
- Holidays increase average sales by approximately 10.7%.
- Sales show a strong relationship with transaction volume.
- Most stores belong to the Low Traffic - Low Ticket segment.
